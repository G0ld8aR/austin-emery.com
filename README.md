# Austin Emery Portfolio Site

Static portfolio website designed for Cloudflare Pages deployment.

## Files
- `index.html` main site markup
- `styles.css` styling and responsive layout
- `script.js` mobile navigation and reveal animations

## Deploy on Cloudflare Pages
1. Push these files into the root of your GitHub repo.
2. In Cloudflare Pages, connect the repo.
3. Use these settings:
   - Framework preset: None
   - Build command: leave blank
   - Build output directory: `/`
4. Save and deploy.

## Easy customizations
- Update project links in `index.html`
- Replace the `mailto:` contact address if needed
- Swap the `AE` avatar block for a real profile photo later
- Add a `resume.pdf` file and point a button to it
